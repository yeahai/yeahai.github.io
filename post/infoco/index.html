我的第一个小项目 信息收集整理

前言

只是技术思路分享,代码完成到数据分类,主要是对架构分析的详细度没到 PHP 7.0.2 这种,对于网站的分析是到PHP、Java 这种大的分类. 如果以后我可以找到 or 我可以自己写细分架构版本的软件，我会更新思路.

我的想法

起初,我也想写一些漏洞扫描器,这些高大上的东西.基于本人对漏洞的理解也只会各种Exp的使用,对其原理理解不深入,暂时对漏扫停止了研究.但,还是想做一些工具对于自己学习编程语言的练习及对安全的学习.

对于漏扫这些很难,那么我就往信息收集这方面靠了靠,只是为了方便信息收集,没有太多的技术点,望大佬斧正.

思路

信息收集是每个测试人员及红队等人员开始工作的第一步,但是对于大量的收集资产进行架构分析一个一个去看,太麻烦了且重复性很高,这一点也不符合我们"程序员"的工作思想,所以,我打算解放双手.

First step

我们查找一个网站的架构,架构版本基本上都是基于一个浏览器插件Wappalyzer或者使用whatweb来进行分析一个网站的架构版本,但是平时一两个网站就还好，如果是几百个网站那就显得很麻烦，且重复.

所以第一步，我打算通过Wappalyzer的API来实现一个Web架构分析器，可以单个网站架构分析，多个网站架构分析.

网站的IP获取: 网站素材没有，所以我在里面写了一个ssl证书crt.sh网站的子域名爬虫 以及网站子域名爆破的子域名收集

ssl证书crt.sh子域名收集

# ssl 证书 查询小爬虫,然后可以通过去重完成目的
class ssl_sub_domain:
    def __init__(self,url):
        self.url = url

    def spider_ssl(self):
        # 要改,单独提出一个函数 ,在全部子域名写入后 再进行去重
        # mys = set()
        url = 'https://crt.sh/?q='+ self.url
        l = []
        response = requests.get(url,head)

        html = response.text
        content = re.compile(r".[^ =]+" + self.url)
        co = content.findall(html)
        for i in co:
            i = i.replace('<TD>', "").replace('<BR>', "\n").strip()
            l.append(i)
            # mys.add(i)
        file_path = "../../search_report_mem/" + self.url + ".txt"
        with open(file_path,'a+',encoding='utf8') as wf:
            for x in l:
                x = x.replace("*.","").replace("'","")
                wf.write(x)
                wf.write('\n')

FUZZ子域名 关于多线程 结尾说 这里的超时注解是当时测试字典大了要扫描很久，设置了一个超时，要是想扫全可以去掉注解.


# 子域名爆破,速度慢,多线程 有尝试过
class fuzz:
    def __init__(self,domain):
        self.domain = domain
    # 这个注解是 方法超时时间(爆破时间太长了) 超过三分钟就停下来
    @func_set_timeout(180)
    def web_sub_domain(self):

        search_memory = "../../python_cms/search_report_mem/" + self.domain +'.txt'
        with open('../../dic/dic.txt','r',encoding='utf8') as rf:
            key = rf.readline().strip()

            while key:
                url ="http://" + key + "." + self.domain
                try:
                    response = requests.get(url,headers=head)
                    if response.status_code == 200:
                        with open(search_memory,'a+',encoding='utf8') as wf:
                            wf.write(url + '\n')
                        print(url)
                except:
                    pass
                key = rf.readline().strip()

通过上面的操作，我们可以获取目标子域名.

Second step

对于获取的子域名数据，我们可以在爬虫的那个网站上看到有很多的重复，所以，我们要去除重复的数据. 这里，我使用的是set数据结构去重，利用set的特性把重复的url进行过滤.

然后，保存到指定文件夹

Third step

API的选取：Wappalyzer API，但是它的API是JavaScript写的，对于Javascript不是很熟悉.然后python的API是一个个人写的，有一些issue，这里我使用的是Tide团队写的TiteFinger 来对网站进行扫描，我把它包裹了一层，修改了里面的一些源码，这篇文章主要是讲工具的思路. TideFinger 没实现多IP扫描，这里我就包裹一下实现多IP的文件扫描,循环调用单个循环.

# 调用 TiteFinger 进行扫描 one
def find_finger(url):
    os_path = "TideFinger.py"
    os.system('python '+ os_path + ' -u ' + url)
# 多次调用TiteFinger 进行扫描, 文件扫描
def file_scan(url_file):
   try:
       with open(url_file, "r", encoding="utf8") as f:
           url = f.readline()
           while url:
               url = "https://" + url
               time.sleep(1)
               find_finger(url)
               url = f.readline()
   except:
       print("file is not exist")

这时候，修改了TideFinger.py中的源代码,让他对扫描的IP地址的资产进行字典转换 {"https://wn.pos.baidu.com": {"banner": " Nginx | P3p", "Cms_name": "Not Found"}} 因为字典对key分了两类，banner，Cms_name

把数据在转换成json 方便以后传输数据给其他的语言或其他工具处理.

我们进行分类通过它的两大类来进行分类,针对每个架构加入网站到指定的banner-“架构”.txt中.

def divide(type):
    if "type".__eq__(type):
        mys = set()
        mys_t = set()

        with open("../Web_Information/Restore.json","r",encoding="utf8")as r :
            data = r.readline()
            datas = json.loads(data,strict=False)

            for i in datas:
                i = eval(i)
                for key, value in i.items():
                    # print(key)
                    # print(value)
                    for m, n in value.items():
                        if ("banner".__eq__(m)):
                            a = n.split("|")
                            for j in a:
                                j = j.strip()
                                mys.add(j)
                                with open("../Web_Information/banner_" + j + ".txt", "a+", encoding="utf8") as mset:
                                    mset.write(key)
                                    mset.write("\n")

                        elif ("Cms_name".__eq__(m)):
                            a = n.split("|")
                            for z in a:
                                if z not in mys_t:
                                    mys_t.add(z)
                                    with open("../Web_Information/cms_" + z + ".txt", "a", encoding="utf8") as mset_t:
                                        mset_t.write(key)
                                        mset_t.write("\n")
    else:
        print("请在-t后面添加type提示进行分类")
