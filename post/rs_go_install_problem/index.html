Rust、Go、Git国内下载慢的Problem

June 12, 2022 in category, subcategory

本文旨在解决Rust、Go下载很慢的问题，经常下载失败,记录Linux和Win下安装Rust、Go的问题

Git

Git Clone慢

Git Config 配置:
git config --global url."https://gitclone.com/".insteadOf https://
git clone https://github.com/tendermint/tendermint.git

使用cgit
cgit clone https://github.com/tendermint/tendermint.git

Git 代理配置

全局代理(不推荐)

# 使用http代理
git config --global http.proxy http://127.0.0.1:58591
git config --global https.proxy http://127.0.0.1:58591

# 使用socks5代理
git config --global http.proxy socks5://127.0.0.1:58591
git config --global https.proxy socks5://127.0.0.1:58591

只对Github代理(推荐)

# socks5
git config --global http.https://github.com.proxy socks5://127.0.0.1:58591
# http 
git config --global http.https://github.com.proxy http://127.0.0.1:58591

取消代理

git config --global --unset http.proxy git config --global --unset https.proxy

Rust

Win

设置临时代理

  1. 在当前 rust-init.exe 路径下打开powershell
  2. 设置临时代理
$ENV:RUSTUP_DIST_SERVER='https://mirrors.ustc.edu.cn/rust-static'
$ENV:RUSTUP_UPDATE_ROOT='https://mirrors.ustc.edu.cn/rust-static/rustup'
  1. 在powershell下执行安装软件 `.\rustup-ini.exe

Linux

设置临时代理

  1. 加入中国科技大学网络镜像代理
$ENV:RUSTUP_DIST_SERVER='https://mirrors.ustc.edu.cn/rust-static'
$ENV:RUSTUP_UPDATE_ROOT='https://mirrors.ustc.edu.cn/rust-static/rustup'

注意以上命令，仅仅在本次终端生效，切换终端，仍然需要在执行一次 2. 执行脚本命令，完成Rust安装

curl https://sh.rustup.rs -sSf | sh
安装成功后，会提示：Rust is installed now .Great!

3. 输入 rust -V 或者 rustup -V 
4. 配置包管理镜像代理
在$HOME/.cargo目录下创建一个名为config的文本文件，其内容为：
```bash
[source.crates-io]
registry = "https://github.com/rust-lang/crates.io-index"
replace-with = 'ustc'
[source.ustc]
registry = "git://mirrors.ustc.edu.cn/crates.io-index"
  1. 安装RSL，RLS（Rust Language Server）
// 喜欢nightly版本的可以尝试最新版rustup 下载，一般我选择稳定版
rustup self update
rustup component add rls rust-analysis rust-src
  1. 安装编译Gcc工具链
sudo apt-get install build-essential
  1. 安装WASM(Web开发）
rustup target add wasm32-unknown-unknown
cargo install wasm-gc
rustup update
  1. racer 自动补全插件，可选，一般用Clion或者Vscode,vim,Emacs编写

cargo install racer

  1. 验证安装成功 rustc –version 出信息则成功

Go

  1. 下载安装包
# 大陆
wget https://studygolang.com/dl/golang/go1.16.5.linux-amd64.tar.gz
# 非大陆
wget https://golang.org/dl/go1.16.5.linux-amd64.tar.gz
tar zxf go1.16.5.linux-amd64.tar.gz -C  /usr/local
  1. 配置环境
vim ~/.bashrc
# 末尾追加如下内容
export GOROOT=/usr/local/go
export GOPATH=$PATH:$GOROOT/bin
export PATH="$PATH:$GOROOT/bin"
go env -w GOPROXY=https://goproxy.cn
  1. 启用环境
source ~/.bashrc
go version
